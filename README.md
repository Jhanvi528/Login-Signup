## MERN Login Signup 

Login and Sign-Up using Reactjs,Node.js + Express and MongoDB made up which can be used as a starting point for another project that needs authentication.

## Features

- Login page with success/error messages and rediresct to homepage
- Signup page with success/error messages
- Protected Homepage route that needs authentication to access
- Persistence achieved using Bcryt used for authentication
- Fully responsive across Desktop, Tablet and Mobile


## Prerequisites

- Node.js
- NPM
- React
- MongoDB Atlas MongoURI
- **.env file with ENV variables**, a .env is provided
